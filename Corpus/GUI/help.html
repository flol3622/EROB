Simulation Parameters Help

General Information

Last Update: 20 April 2021

Author: Silke Verbruggen

Result File Name

Specify the name for the result file.

name = 'Test'

Simulation Length

Define the simulation length in days and the starting year.

nday = 365

year = 2021

Number of Buildings

Define the number of buildings to simulate.

nBui = 1

Household Members

Choice to not define anything, or to define the number of persons, or define the employment types, or even more detailed define occupancy types.

npers = 0 (not predefined)

Members: Choose from FTE (full-time employed), PTE (part-time employed), Unemployed, Retired, School (U18), Student.

members = [] (if not predefined)

Example: members = ['FTE', 'FTE', 'Retired', 'School']

Occupancy Profiles (Clusters)

Define occupancy profiles for each weekday.

Choices: nightshift (1), early shift (2), short absence (~3h) (3), dayshift (4), half day absence (5), long absence (6), always present (7).

clusters = [] (if not predefined)

Example: clusters = [{'mon':4,'tue':4, 'wed':4, 'thu':4, 'fri':6, 'sat':3, 'sun':7}]

Bedrooms

Define the number of bedrooms or the bedroom assignment.

nbedr = 0 (not predefined)

If both members and number of bedrooms are defined, you can define who sleeps together in one room.

BEDR = [] (if not predefined)

Example: BEDR = [0,0,1] (for household ['FTE','FTE','School'], where the two FTE persons sleep together and the child sleeps in a separate bedroom)

Appliances

Define appliances from the list defined in appliances.py.

apps = [] (if not predefined)

Building Properties

Define some building properties. If unknown, set to -1.

Ventilation System: VentS = -1 (0 = no system, 1 = exhaust ventilation, 2 = balanced ventilation)

Dwelling Type: DW = -1 (1 = apartment, 2 = house)

Year Built: YearBuilt = -1

Occupancy and Activity Profiles on File

Indicators if the occupancy and activity profiles are already defined on files (1) or not (0).

OccONFILE = 0

ActONFILE = 0

Heating Profile

Define the heating profile. If not predefined, set to -1.

shtype = -1

Define zones to be heated.

shrooms = []

Window Use Habits

Define the habits for each room and leaving and sleeping, or define household habit in winter and seasonality coherence.

habits = [] (unknown)

Household Habit: HHhabit = -1

Seasonality Coherence: SeCo = -1
